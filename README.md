
# Manage MyDay - 个人日程管理工具

Manage MyDay 是一个基于 PyQt6 开发的现代化桌面日程管理应用。它结合了日历视图、任务清单、分类标签和数据统计功能，旨在为您提供高效、直观的个人事务管理体验。

## ✨ 主要功能

* **直观的日历视图**
  * **月视图浏览** : 轻松切换月份和年份，概览全月安排。
  * **农历支持** : 内置农历日期和中国传统节日显示（需 `zhdate` 库）。
  * **任务标记** : 日历上通过颜色条和星标直观展示每日最重要的任务和优先级。
  * **今天跳转** : 一键快速回到当前日期。
* **强大的任务管理**
  * **任务增删改** : 便捷地添加新任务，支持编辑内容、详细描述和修改状态。
  * **优先级系统** : 支持普通、重要(★)、紧急(★★)、非常紧急(★★★)等优先级设定。
  * **状态追踪** : 任务状态包括“待完成”、“进行中”、“已完成”、“搁置”。
  * **详细描述** : 支持为每个任务添加详细的备注说明。
* **灵活的标签分类**
  * **预设标签** : 内置工作、生活、学习、健康、其他等常用分类。
  * **自定义标签** : 支持创建带有自定义颜色的新标签。
  * **侧边栏筛选** : 通过侧边栏复选框快速筛选特定标签的任务。
* **悬浮便签模式 (Mini Mode)**
  * **桌面置顶** : 将应用最小化为一个小巧的黄色便签，始终显示在桌面最顶层。
  * **今日聚焦** : 仅显示今日待办事项，保持专注。
  * **快速操作** : 双击便签条目即可快速切换任务完成状态。
  * **进度展示** : 便签内置进度条，直观显示今日任务完成度。
* **数据管理与统计**
  * **数据统计** : 饼图/数据面板展示任务总数、完成率及重要任务数量。
  * **备份与恢复** : 支持本地数据库一键备份。
  * **导入导出** : 支持将数据导出为 JSON 格式或从 JSON 导入，方便数据迁移。
* **现代化界面**
  * **暗色模式** : 精心设计的深色主题 (Dark Mode)，护眼且美观。
  * **响应式布局** : 支持拖拽调整侧边栏和任务面板的宽度。
  * **置顶功能** : 支持主窗口置顶显示。

## 🛠️ 安装与运行

### 1. 环境要求

请确保您已安装 Python 3.8 或更高版本。

### 2. 安装依赖库

在项目根目录下运行以下命令安装所需的第三方库：

```
pip install PyQt6 zhdate
```

* `PyQt6`: 用于构建图形用户界面。
* `zhdate`: (可选) 用于显示农历日期。

### 3. 运行应用

运行 `main.py` 启动程序：

```
python main.py
```

## 📂 项目结构

* `main.py`: 应用程序的主入口，包含 UI 逻辑、事件处理和自定义控件（如日历、便签）。
* `task_manager.py`: 负责后端数据逻辑，包括 SQLite 数据库操作（增删改查）、任务对象定义。
* `mac_style.qss`: 样式表文件，定义了应用的深色主题外观。
* `myday.db`: (自动生成) SQLite 数据库文件，存储所有任务和标签数据。
* `ico_image/`: (自动生成) 用于缓存下载的图标资源。
* `backups/`: (自动生成) 用于存放数据库备份文件。

## 💡 使用技巧

1. **右键菜单** : 在任务列表上点击右键，可以快速修改任务状态、设置优先级或删除任务。
2. **悬浮便签** : 点击侧边栏的“悬浮便签”按钮进入极简模式，右键点击便签可调节透明度。
3. **拖拽调整** : 界面主要区域的分割线支持左右拖拽，您可以根据需要调整日历和任务列表的宽度。
4. **高级搜索** : 侧边栏的“高级搜索”功能支持组合条件（关键字、日期范围、标签、优先级）筛选任务。

## 📝 开发说明

本项目使用 SQLite 作为本地存储，数据安全且易于迁移。所有图标资源均来自网络（首次运行时会自动下载缓存）。
